# Rubik’s Cube Solver

GPGPU Final Project Team 2

M10615005 & M10615004

## 目標
1. 解開任意大小的魔術方塊(複雜度)
1. 越快找出答案越好(時間)
1. 答案越少步越好(步驟)


## Milestone
- [x] 魔術方塊模擬器
- [x] 可以解開2x2x2
- [x] 可以解開3x3x3
- [x] 友善的GUI介面
- [ ] 可以解開NxNxN
- [ ] 不超過God’s Number的解
- [x] 3D視角的GUI介面
- [ ] 可以用動畫展示解答
- [ ] 實作其他方法用來比較
